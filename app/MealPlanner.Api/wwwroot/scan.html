Новий use case

Сканування QR/штрихкоду продукту

Ця сторінка демонструє виклик Web API через JavaScript: прив'язку коду до продукту, сканування коду з камери та запис історії сканувань.

Назад до API-клієнта Swagger

1. Прив'язати код до продукту

Для демонстрації можна ввести будь-яке значення коду вручну, наприклад 482000000001. Якщо продукт має реальний штрихкод або QR-код, можна ввести або сфотографувати його.

2. Сканувати код

Якщо браузер не підтримує BarcodeDetector, сторінка використовує ZXing fallback. На телефоні найзручніше натиснути «Сфотографувати код».
